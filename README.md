# Detect_OSystem

Script en `python` para saber la información de una maquina victima, ingresando la `ip`

## Requisitos

* Python
* Linux/Unix

## Instrucciones

1. Ejecutar el archivo `main.py`

~~~console
python3 main.py

[+] Enter the ip address: 10.10.10.10

[+] ip: 10.10.10.10
[+] ttl: 128
[+] os: Windows
[+] open ports: [ 21, 22, 80, 587 ]
~~~
